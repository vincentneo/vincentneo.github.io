Avenue

macOS App

Avenue is a free, no-frills GPX file viewer for macOS Sierra 10.12 and above. Designed for ease-of-use.

Features include a mini map for easier navigation of large tracks, third-party map support, and more.

It relies on CoreGPX and is a fork of iOS-Open-GPX-Tracker.

Distributed under GPLv3 license.





CoreGPX

Multi Platform Library

CoreGPX is a library for easier parsing and creating of GPX files for iOS, macOS and watchOS.

Currently supports GPX 1.1 standards, with GPX 1.0 support arriving in the future.

Available on CocoaPods, Carthage or Swift Package Manager.

Distributed under MIT license.
